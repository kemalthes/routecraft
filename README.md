# Routecraft 
Routecraft - сайт для создания и просмотр маршрутов.
## Возможности
### Для пользователей
- Регистрация с отправкой письма на почту
- Логин по почте и паролю
- Функция восстановления и смены пароля
- Добавление маршрутов-черновиков
- Возможность редактировать метаинформацию на них
- Взаимодействие с динамической картой
- Добавление в избранное
- Возможность оставить комментарии
### Для админов
- Возможность управлять пользователями
- Публикация маршрутов-черновиков и их одобрение
- Модерация комментариев
Дополнительно: есть ИИ-семантический поиск.

### Запуск:

1. Клонируйте этот репозиторий
```bash
git clone "https://github.com/kemalthes/routecraft.git"
```
2. Заполните .env, пример необходимых env полей находится в .env.sample
3. Выполните 
``` bash
docker compose up
```
